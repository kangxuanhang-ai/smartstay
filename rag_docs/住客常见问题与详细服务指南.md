# 住客常见问题与详细服务指南

## 一、入住与退房

### 入住流程
1. 前台出示身份证件办理登记
2. 支付押金（信用预授权¥3,000）
3. 领取房卡（支持刷脸/房卡/手机NFC三种方式）
4. 前台可协助搬运行李至房间
5. 首次入住需修改初始密码（默认123456）

### 退房流程
1. 前台结算房费及消费
2. 退还押金（预授权自动解冻，1-3个工作日到账）
3. 归还房卡
4. 如需发票，前台即时开具

### 提前入住
- 14:00前到店：若房间已清洁完毕可提前入住（视房态）
- 若房间未准备好，可免费寄存行李，先使用酒店设施
- 提前至10:00入住加收半天房费，提前至06:00入住加收全天房费

### 延迟退房
- 标准退房时间：12:00
- 延迟至14:00：加收半天房费
- 延迟至18:00：加收全天房费
- 18:00后未退房视为次日入住，扣除全天房费
- 金卡会员免费延迟至14:00，钻石卡会员可灵活安排

### 取消政策
- 入住当天18:00前免费取消，全额退还
- 18:00后取消扣除首晚房费
- No-show（未到店）扣除首晚房费
- 节假日（春节、国庆）预订不可取消

## 四、智能客房系统详细操作

### 灯光控制
- 开关方式：墙上面板 / 手机App / AI语音
- 可控区域：客厅灯、卧室灯、床头灯、阅读灯、夜灯
- 场景模式：
  - 阅读模式：床头灯亮起，其他灯关闭
  - 睡眠模式：全部灯光关闭，夜灯开启50%亮度
  - 全亮模式：所有灯光100%亮度
  - 迎宾模式：客厅灯+卧室灯50%亮度
- 语音指令示例：
  - "打开客厅灯" / "关闭卧室灯"
  - "调暗灯光到30%"
  - "打开阅读模式"
  - "全部关灯"

### 窗帘控制
- 控制方式：墙上面板 / 手机App滑动条
- 调节范围：0%-100%无级调光
- 自动模式：早晨7:00自动开启至30%，晚上22:00自动关闭
- 语音指令示例：
  - "打开窗帘" / "关闭窗帘"
  - "窗帘开到一半"
  - "把窗帘打开30%"

### 空调控制
- 模式：制冷 / 制热 / 送风 / 除湿
- 温度范围：16°C - 30°C（0.5°C步进）
- 自动节能：无人时自动切换（夏季26°C，冬季20°C）
- 语音指令示例：
  - "调低空调到22度"
  - "打开制冷模式"
  - "调高两度"
  - "关闭空调"

### 设备控制
- 电视：语音指令"打开电视"/"关闭电视"/"换到CCTV1"
- 音响：支持蓝牙连接，语音指令"播放音乐"
- 热水器：自动加热，无需手动操作
- 所有控房操作均有500ms防抖，防止连续触发

## 九、安全与应急详细指南

### 火灾应急
1. 发现火情立即拨打119，并通知前台（内线8888）
2. 按下最近的火灾报警器
3. 沿应急指示灯从楼梯撤离（勿使用电梯）
4. 到达1F大堂集合点清点人数
5. 酒店每层走廊两端配备干粉灭火器和消防栓
6. 每间客房配备烟雾报警器和自动喷淋系统

### 医疗急救
1. 立即拨打120
2. 通知前台（内线8888），安保人员2分钟内到达
3. AED自动体外除颤器位于1F大堂前台右侧红色急救箱上方
4. 各楼层配备急救包（创可贴、碘伏、纱布、三角巾）
5. 酒店工作人员持有急救证书

### 地震应急
1. 迅速躲到桌子下方或楼梯间
2. 远离玻璃幕墙、吊灯、书架等易倒物品
3. 地震停止后沿楼梯有序撤离
4. 各楼层楼梯间为抗震加固区

### 贵重物品安全
- 电子保险箱：位于衣柜内，设置6位数密码
- 建议将贵重物品存入保险箱
- 前台提供免费保险箱寄存服务（大件物品）
- 如有物品遗失，请联系前台或拨打安保内线8888
