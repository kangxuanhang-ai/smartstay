# 智宿云大酒店服务完全手册

## 1. 酒店概况

智宿云大酒店位于北京市朝阳区建国路100号，地处CBD核心商圈，开业于2020年，总客房数120间。酒店以"智能、舒适、人文"为经营理念，所有客房配备SmartStay智能家居系统。

酒店共6层：B1层为SPA水疗和棋牌室、1F为接待大堂和中西餐厅、2F为商务中心和会议室、3-5F为客房区（大床房/双床房/套房）、6F为行政酒廊、无边泳池和24H健身房。

周边地标距离：中央电视台总部大楼0.8km、国贸商城1km、三里屯太古里1.5km、故宫博物院3km、天安门广场3.5km、颐和园12km、长城（八达岭）68km。

## 2. 前台服务

接待大堂位于1F正门入口，24小时前台服务。入住时间为14:00，退房时间为12:00。所有住客需凭有效身份证件登记入住。押金信用预授权为¥3,000元，可通过微信、支付宝、银联或现金支付。

前台提供以下免费服务：行李寄存、叫醒服务、外币兑换（美元/欧元/日元/港币）、叫车服务、旅游咨询、快递收发。另外提供会议室预订、接送机预约（奔驰商务车接送机¥500/单次，需提前2小时预约）、门票代购等增值服务。

## 3. 客房设施

酒店共有三种房型，所有房间标配智能家居控制系统、55英寸4K智能电视、minibar、电子保险箱、高速WiFi、独立淋浴间、吹风机、浴袍和拖鞋。

大床房（45㎡）：配1.8m×2m特大床一张，位于3-4层，适合1-2人入住。基价¥300/晚。

双床房（55㎡）：配1.5m×2m标准床两张，位于3-4层，适合2-3人入住。基价¥350/晚。

套房（80㎡）：独立客厅+卧室，1.8m×2m特大床，配胶囊咖啡机、行政酒廊权益、独立浴缸、双洗手台，位于5层。基价¥600/晚。

## 4. 智能客房系统

所有客房均配备SmartStay智能家居控制系统，支持以下功能：

灯光控制：可通过墙上面板或手机App独立控制客厅灯、卧室灯、床头灯的开关，支持场景模式（阅读模式、睡眠模式、全亮模式）。

窗帘控制：电动窗帘支持0%到100%无级调光，可通过面板或手机滑动条任意调节开合程度。早晨7:00自动开启至30%，晚上22:00自动关闭。

空调控制：支持制冷和制热双模式，温度可在16°C至30°C之间以0.5°C为步进调节。系统自动检测房间是否有人，无人时自动切换至节能模式（夏季26°C，冬季20°C）。

语音控制指令：对AI管家说"打开客厅灯""调低空调到22度""关闭窗帘"即可完成操作。所有控房操作均挂载500ms防抖机制，防止连续触发造成系统负载过高。

## 11. 安全与应急

前台24小时紧急联系电话：13800000002。内部安保24小时巡逻。

通用紧急电话：急救120、火警119、公安110。

酒店安保内线：8888（24小时值班）。

消防设施：每层走廊两端配备干粉灭火器和消防栓，每间客房配备烟雾报警器和自动喷淋系统。各楼层电梯间张贴消防逃生路线图。火灾时请勿使用电梯，沿应急指示灯从楼梯撤离至1F大堂集合。

急救设备：AED自动体外除颤器位于1F大堂前台右侧红色急救箱上方，培训合格人员24小时在岗。各楼层配备急救包（含创可贴、碘伏、纱布、三角巾）。

地震避险：北京市属于地震设防区域，各楼层楼梯间为抗震加固区，地震时请迅速进入楼梯间避险，远离玻璃幕墙和吊灯。

极端天气：遇极端天气（暴雨红色预警、暴雪等），前台将提供免费延迟退房服务，并为被困住客提供免费方便面和热水。
